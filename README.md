# ИС СА — интерактивный прототип

Информационная система предварительной проработки решений о строительстве и ремонте объектов (Московская область). Интерактивный прототип для отработки удобства работы пользователей (UX).

## Стек

React 19 + TypeScript + Vite + Tailwind CSS + shadcn/ui

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Функциональные сценарии прототипа

- Реестр объектов: поступление из внешних ИС после геоанализа, ручное добавление, фильтры и поиск
- Рассмотрение объекта координатором и назначение ведомств для проверок
- Выездные проверки ведомств: фото, комментарии, заключение (согласовано / отклонено)
- Автоматическое формирование итогового заключения по итогам всех проверок
- Дорожные карты объектов: задачи, плановые/фактические даты, ответственные, статусы, диаграмма Ганта
- Объединение объектов в проекты со сводной дорожной картой
- Личные кабинеты ролей: координатор, сотрудник ведомства, руководитель
- Уведомления о новых объектах и событиях по задачам

Данные демонстрационные.

# История изменений

## [1.1.0] — 2026-07-17

### Добавлено
- **Раздел «Текущая работа»** — дашборд по согласованным объектам: исполнение решений после итогового заключения.
  - Статистика по решениям: сохранение (строительство / капитальный ремонт / текущий ремонт) и закрытие (передача / снос).
  - Данные по финансированию: план / освоение, разрез по источникам средств.
  - Детализация по типам объектов (школа, детский сад, поликлиника и т.д.) при выборе вида решения.
  - Статистика по муниципалитетам: решения, финансирование, готовность, просроченные задачи.
  - Таблица «Ход работ по объектам» с готовностью, освоением и просрочками.
- **Решение по объекту**: модель данных, оформление и редактирование в карточке объекта (вид решения, дата, источник, плановое и освоенное финансирование).
- **Тип объекта (категория)** у всех объектов, в т.ч. в диалоге ручного добавления и в карточке.
- Раздел **«Общая картина»** — дашборд процесса согласования: воронка, SLA по этапам, скорость ведомств, зависшие проверки, динамика по месяцам, округа и отрасли, «красная зона».
- Дата назначения проверки и муниципальный округ в модели данных.

### Изменено
- Демонстрационные данные расширены до 20 объектов в 14 муниципалитетах с историей с февраля 2026.

## [1.0.0] — 2026-07-16

Первая версия интерактивного прототипа ИС СА:
- реестр объектов с фильтрами и ручным добавлением;
- рассмотрение объекта координатором и назначение проверок ведомствам;
- выездные проверки с фото, комментариями и заключением;
- автоматическое итоговое заключение;
- дорожные карты объектов (план/факт), проекты со сводной дорожной картой;
- личные кабинеты и уведомления;
- переключатель ролей (координатор / сотрудник ведомства / руководитель).
